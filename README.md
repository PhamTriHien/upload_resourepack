# upload_resourepack

Resource pack files for the Purpur server.

Recommended usage:

1. Push the `.zip` files to GitHub.
2. Create a GitHub Release.
3. Upload the `.zip` files as release assets.
4. Use the release asset URL in ForcePack.

Example ForcePack URL:

```text
https://github.com/PhamTriHien/upload_resourepack/releases/download/v1/enderman.zip
```
